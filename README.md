# Sink_the_Fleet (Python)

### Bienvenidos al juego de **Hundir la flota**

Se trata de una version de código Python en la que podrás enfrentarte a la IA.
Características:
- Menú de juego 
- Nivel de dificultad medio (la IA verifica su tirada 1 vez antes de disparar)
- Tamaño del tablero elegible
- Coordenadas elegibles por el usuario y aleatorias para la IA
- Flota de barcos aleatoria según tamaño del tablero
- Opción de volver a jugar sin tener que ejecutar el codigo

### Instrucciones

1. Clona el repositorio: https://github.com/CPasData/Sink_the_Fleet.git. Necesitarás los ficheros *main.py* y *utils.py*
2. Ejecuta el fichero *main.py*.
3. Sigue las instrucciones que te aparecen en pantalla.
4. ¡Disfruta!
